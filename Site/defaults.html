There are 10 defaults animations preloaded with Ixhibition, which use the naming convention "ixb_#", where # is a number from 1 to 10. These defaults can be used using the "loadOption" function.
The following show a demo for each option.

ixb_1 Stack (no fade)
ixb_2 Vertical simple
ixb_3 Vertical-reverse simple
ixb_4 Horizontal simple
ixb_5 Horizontal-reverse simple
ixb_6 Stack with fade
ixb_7 Fade, with fade-out drop
ixb_8 Slide focus, with slight zoom
ixb_9 Squash to focus
ixb_10 Fold-in Fold-out
